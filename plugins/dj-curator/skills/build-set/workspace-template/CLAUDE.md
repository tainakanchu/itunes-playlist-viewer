# 選曲ワークスペース (Crateforge)

このディレクトリは **DJ 選曲の作業場**。ここで Claude Code を開き、`dj-curator` プラグインの
`/dj-curator:build-set` で叩き台を作る。Crateforge アプリの内蔵 API（既定
`http://127.0.0.1:8787`）を有効にしておくこと。

---

## 選曲の基本方針 (ここを自分用に書き換える)

### 母集団 (どこから選ぶか)
- 既定の母集団は **★3 以上**（API では `ratingMin=60`）。
- 基本はライブラリ全体から。ただし下記の好み/除外を効かせる。

### 好きな方向 / よく使う
- 例: Deep House / Melodic Techno / Disco Re-edit を軸に。
- 例: 120–126 BPM をホームに、夕方は energy 低めから。

### 避けたい
- 例: 露骨な EDM ビッグルーム、年代が古すぎる音質のもの。
- 例: 過去 3 ヶ月のセットで使った曲はなるべく避ける（`sets/` 参照）。

### DJ の色 / セットの作法
- 例: 序盤は歌もの少なめ、ピークで 1〜2 曲だけアンセム。
- 例: キーは無理に詰めないが、隣同士は Camelot 互換だと嬉しい。

---

## 運用メモ
- 叩き台のプレイリストは Crateforge アプリに作られる。**曲順はアプリ上で詰める**。
- セットの記録は `sets/<日付>-<イベント>.md` に残す（重複回避と振り返り用）。
- フライヤー画像やテーマメモは `inputs/` に置くと投げやすい。
- 別ポートで動かしている場合は環境変数 `CRATEFORGE_API` を設定。
