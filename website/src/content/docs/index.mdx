## Crateforge とは

**Crateforge** は Tauri 2 + React 19 + Vite 6 + Rust 製の、爆速 iTunes 風 音楽管理デスクトップアプリです。
`iTunes Library.xml` の取り込み / 書き出し、ローカルファイル取り込み、CD リッピング、プレイリスト編集、ローカル再生に加え、
DJ 向けの解析（BPM / Key (Camelot) / Energy）・類似度選曲・フォーマット変換・スマートプレイリストまで、**このアプリ単体で完結**します。

> 旧名は「iTunes Playlist Viewer」です。crate（掘る・集める）+ forge（鍛える・組み上げる）で、解析して選曲しセットを鍛えるという中身を表す名前にリブランドしました。

## 対応 OS

- **Windows** — `.exe` / ポータブル `.zip` / `.msi` / セットアップ `.exe`。アプリ内の自動更新に対応（exe をその場で差し替え）。
- **macOS** — **未署名** の `.dmg`（Apple Silicon / Intel）。初回起動は Gatekeeper を回避する必要があります。
- **Linux** — `.AppImage`（単一ファイル）と `.deb`。

> アプリ内の自動更新は **Windows 専用** です。macOS / Linux は手動更新になります。
