曲目表格與右側欄都能依你的使用習慣細部調整。設定會被保存。

> 截圖稍後補上

## 欄的自訂

- **調整欄寬** — 拖曳標頭右緣即可變更欄寬（也能把 Genre / 標籤欄拉寬到完整顯示）。
- **欄的排序** — 直接拖曳標頭，或在欄位挑選器（自訂選單）中以指標操作排序。
- **切換顯示的欄** — 在欄位挑選器中切換欄的顯示 / 隱藏。
- **曲目編號欄** — 可顯示兩種：專輯內的曲目編號「Track #」，以及依目前顯示順序的流水號「No.」。

## 列高、封面、右側欄

- **列高** — 以滑桿在 32〜64px 範圍內調整。
- **封面圖（封面）尺寸** — 可從 無 / 小（20px）/ 中（28px）選擇。
- **List / Covers 檢視** — 以工具列的分段控制切換。Covers 會把同一張專輯彙整為一張卡片，點選後就地展開其中的曲目清單。
- **右側欄的顯示 / 隱藏** — 可用切換鈕隱藏。

被省略顯示的儲存格（曲名 / 專輯 / 專輯演出者），只要滑鼠懸停就會以工具提示顯示全文。

## CJK 字體差異吸收搜尋

在搜尋框、內建 API、智慧型播放清單中，會將 **繁體字 / 簡體字 / 日文漢字與假名（平假名↔片假名）/ 全形半形 / 英文大小寫** 正規化，
無論用哪種字體輸入都能橫跨命中。強度可在設定中從 **關閉 / 輕量 / 標準** 選擇。

搜尋可用 `/` 取得焦點，將查詢以空格分割後，把各 token 以 AND 結合
（會比對 名稱 / 演出者 / 專輯 / 專輯演出者 / 類型 / 註解 之任一）。
也可用 `bpm:120-128` / `key:8A` / `energy:60-100` 這類[解析值](../dj-analysis/)來篩選。

## 鍵盤快捷鍵

按 `?` 鍵可顯示快捷鍵一覽的覆蓋層。主要項目如下。

### 播放

| 按鍵 | 操作 |
|---|---|
| `Space` | 播放 / 暫停 |
| `Enter` | 播放選取的曲目 |
| `J` | 上一首 |
| `K` | 下一首 |
| `Shift` + `←` | 倒退 5 秒 |
| `Shift` + `→` | 快轉 5 秒 |
| `S` | 切換隨機播放 |
| `R` | 切換重複播放 |
| `Ctrl` + `↑` | 提高音量 |
| `Ctrl` + `↓` | 降低音量 |

### 導覽與搜尋

| 按鍵 | 操作 |
|---|---|
| `/` | 焦點移至搜尋 |
| `Ctrl` + `F` | 焦點移至搜尋 |
| `Ctrl` + `L` | 返回音樂庫（清除搜尋） |
| `Esc` | 離開搜尋 / 輸入、關閉對話框 |

### 清單操作

| 按鍵 | 操作 |
|---|---|
| `↑` / `↓` | 上下移動選取 |
| `Shift` + `↑` / `↓` | 擴展選取範圍 |
| `Ctrl` + `A` | 全選 |
| `Ctrl` + `I` | 編輯選取的曲目（Get Info） |
| `≣`（應用程式鍵） | 內容選單（以焦點列為基準顯示） |

### 說明

| 按鍵 | 操作 |
|---|---|
| `?` | 顯示此快捷鍵一覽 |
