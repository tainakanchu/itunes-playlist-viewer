可將曲目轉換為其他音訊格式（使用 ffmpeg）。會沿用標籤與內嵌封面圖，並顯示進度。

> 截圖稍後補上

## 進行轉換

1. 選取曲目（可多選），**右鍵 →「Convert to…」**。
2. 選擇 **格式、位元率、輸出位置**。
3. 如有需要，啟用「轉換後加入音樂庫」。
4. 開始轉換後會顯示進度。

對話框可用 `Esc` 關閉，並支援初始焦點與 `Enter` 確認。

## 支援的格式

| 格式 | 類別 | 備註 |
|---|---|---|
| **MP3** | 失真 | 可指定位元率 |
| **AAC** | 失真 | 可指定位元率 |
| **Opus** | 失真 | 可指定位元率 |
| **FLAC** | 無損 | |
| **ALAC** | 無損 | |
| **WAV** | 未壓縮 | |

失真格式（MP3 / AAC / Opus）可選擇 **位元率**。

## 輸出位置與加入音樂庫

- 可指定 **輸出資料夾**。
- 啟用「加入音樂庫」後，會直接將轉換後的檔案匯入音樂庫
  （轉換後的檔案會 **原地保留在指定的輸出資料夾**，整理目標資料夾（音樂庫根目錄）的配置規則不會套用至轉換）。

轉換時，音樂庫的標籤與內嵌封面圖會沿用至新檔案。

## 關於 ffmpeg

轉換 **以外部程序的方式使用 ffmpeg**。

- **Windows** — 不隨附 ffmpeg，而是 **在首次使用時自動取得並快取**。
  解析順序為 `%LOCALAPPDATA%` 的快取 →（舊有的隨附資源）→ PATH → 都沒有時再從上游（BtbN）下載。即使更新應用程式，ffmpeg 也會保留。
- **macOS / Linux** — 使用 **PATH 上的 ffmpeg**（請另行安裝）。

:::note
由於 ffmpeg 採 GPL 授權，因此避免隨附，改以 CLI 形式作為外部程序使用。
:::
