**智慧型播放清單** 是依條件（規則）自動決定內容的播放清單。
每次開啟都會評估整個音樂庫，因此始終保持最新狀態。

> 截圖稍後補上

## 建立與編輯

- 用側邊欄的 **🎛 按鈕** 新增。
- 既有的智慧型播放清單可用 **右鍵 → 編輯規則** 來編輯條件。

## 條件（規則）

在編輯器中，以下列組合定義規則。

- **欄位 / 運算子 / 值** — 1 列為 1 個條件。
- **全部 / 任一符合** — 以 AND / OR 結合多個條件。
- **排序** — 結果的排列順序。
- **上限（件數限制）** — 從開頭限制為 N 件。

可用於條件的欄位，除了基本中繼資料外，還包含 **解析值**。

- **BPM** / **Key (Camelot)** / **Energy**
- **播放次數** / **略過次數** / **評等** / **最後播放時間**
- 名稱 / 演出者 / 專輯 / 專輯演出者 / 類型 / 年份 等

### 輸入輔助

- **日期條件** 可用原生的日期輸入指定。
- 類型等字串條件適用 [CJK 字體差異吸收](../customize/)（繁體字 / 簡體字 / 日文漢字與假名的正規化）。

## 資料夾階層

智慧型播放清單與一般播放清單相同，可放置於側邊欄的 **資料夾階層** 中。
資料夾可點選摺疊 / 展開，且該狀態會被保存，重新啟動後仍維持。

:::note
匯入的 iTunes XML 之智慧型播放清單條件（`Smart Info`）不會被保留。
請在 Crateforge 端重新建立規則。
:::
