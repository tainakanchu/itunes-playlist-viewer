## 什麼是 Crateforge

**Crateforge** 是一款以 Tauri 2 + React 19 + Vite 6 + Rust 打造的極速 iTunes 風格音樂管理桌面應用程式。
除了 `iTunes Library.xml` 的匯入 / 匯出、本機檔案匯入、CD 擷取、播放清單編輯、本機播放之外，
還涵蓋了 DJ 取向的解析（BPM / Key (Camelot) / Energy）、相似度選曲、格式轉換、智慧型播放清單，**只用這一個應用程式就能全部完成**。

> 舊名為「iTunes Playlist Viewer」。以 crate（挖掘、蒐集）+ forge（鍛造、組建）重新命名，象徵透過解析來選曲、鍛造一套 set 的核心精神。

## 支援的作業系統

- **Windows** — `.exe` / 可攜式 `.zip` / `.msi` / 安裝程式 `.exe`。支援應用程式內自動更新（就地替換 exe）。
- **macOS** — **未簽署** 的 `.dmg`（Apple Silicon 原生）。首次啟動需要繞過 Gatekeeper。
- **Linux** — `.AppImage`（單一檔案）與 `.deb`。

> 應用程式內的自動更新 **僅限 Windows**。macOS / Linux 需手動更新。
